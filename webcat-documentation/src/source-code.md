WEBCAT aims to be fully transparent and reproducibile. As such, all its components are publicly developed and available under open source licenses.

### Repositories
WEBCAT components are distributed in several different repositories, under the [freedofpress](https://github.com/freedomofpress) Github organization.

 - [webcat-spec](https://github.com/freedomofpress/webcat-spec) - Text-based protocol and format specifications
 - [webcat](https://github.com/freedomofpress/webcat) - Browser extension
 - [webcat-cli](https://github.com/freedomofpress/webcat-cli) - Comamnd line utilities for developers and website administrators
 - [webcat-infra-chain](https://github.com/freedomofpress/webcat-infra-chain) - Census-based enrollment distributed system
 - [webcat-docs](https://github.com/freedomofpress/webcat-docs) - This documentation website

### Demo
